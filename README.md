Thor
====

My pdf prototype tool

##Prerequisite

 1. [poppler][1]
 2. [poppler-data][2]

## Install

    python setup.py develop

## Test

 - Auto-test (watching .py)
  - `run_pyspecs.py`
 - Run all tests
  - `pyspecs_.py`
